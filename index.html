Login Animate | Login Animado

O projeto que desenvolvi é uma tela de registro e login com uma transição suave entre as páginas. Com uma abordagem elegante e amigável, criei um sistema intuitivo que permite aos usuários transitarem entre as páginas com apenas um clique. Utilizando HTML, CSS (keyframes para animações) e JavaScript, apliquei técnicas de manipulação do DOM e estruturei o código seguindo o padrão de nomenclatura camel case.

My Computer | E-commerce

O projeto "MyComputer E-commerce" desenvolvido em HTML, CSS e JavaScript é uma plataforma de comércio eletrônico que oferece aos usuários uma experiência navegação e adição de produtos no carrinho de compras. Esse sistema de carrinho em JavaScript e navegação intuitiva entre todas as páginas, os clientes podem adicionar produtos ao carrinho e dinamicamente efetuar o calculo de suas compras.

Registration Form | Formulário de registro

No projeto, você encontrará um formulário e uma tabela com uma interação intuitiva entre eles. O usuário preenche os campos do formulário e os dados são validados por meio de JavaScript antes de serem armazenados na tabela. A tabela é preenchida com os dados fornecidos, e o design visual é agradável, organizado, limpo e suave. Além disso, os usuários podem interagir com a tabela para visualizar e remover dados presentes nela.

Hard skills

Tenho expertise em criar estruturas sólidas e semânticas, utilizando as melhores práticas do setor para desenvolver páginas web eficientes e acessíveis. Estou constantemente atualizado com as últimas tendências e recomendações do W3C, para entregar projetos de alta qualidade e proporcionar uma ótima experiência aos usuários.

Animações CSS para adicionar movimento e interatividade às páginas web. Compreendo as propriedades-chave, como @keyframes, animation, transition, e como aplicá-las. Familiarizado com a sintaxe básica de JavaScript, incluindo variáveis, tipos de dados, operadores e estruturas de controle, condicionais e loops, como definir e usar funções em JavaScript, incluindo parâmetros, argumentos, escopo e retorno de valor.
